基本三角函数的计算：

  • 角的正弦值 = 对边 / 斜边
  • 角的余弦值 = 邻边 / 斜边
  • 角的正切值 = 对边 / 邻边

角度制与弧度制的相互转换：

  • 弧度 = 角度 * Math.PI / 180
  • 角度 = 弧度 * 180 / Math.PI
